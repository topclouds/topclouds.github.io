How Does It Work?

Every september, we analyzed the top web sites to determine who the top hosting providers are. The process is fairly simple:

  1. We download the top 1M website file from Alexa and we extract the top 100K.
  2. We perform a DNS lookup of the web site listed in the file to get the IP address of the main website, then we do a reverse DNS lookup to get the owner of the IP.
  3. We "curl" the website to get the web server information.
  4. We compare the IP address with the list of known IPs for each of the cloud providers.
  5. We process and present the results.

Disclaimer

The data produced and presented from the above methodology is as is and we make no interpretation from it. You should use your own judgement and knowledge to interpret and draw conclusions from the data.
